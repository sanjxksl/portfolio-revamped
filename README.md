# Portfolio

Personal portfolio website showcasing my work at the intersection of product management, data analytics, and design.

## Tech Stack

- HTML, CSS, JavaScript (vanilla)
- No frameworks or dependencies

## Local Development

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js
npx http-server
```

Then open `http://localhost:8000` in your browser.
